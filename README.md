# Luno Institutional -- Information Architecture

A single-page HTML knowledge base documenting the information architecture for Luno's institutional platform. Covers user personas, role-based permissions, platform configurations, and end-to-end workflows for every institutional role.

**Live site:** https://bstricker91.github.io/institutional-ia-knowledge/

## What's Inside

- **Foundations** -- User personas, role matrix, and platform configuration models
- **Permissions** -- Role-based permissions mapped across all institutional roles
- **Workflows** -- Step-by-step flows for Designated Members, IFAs, Portfolio Officers, Governance & Compliance, Principals, Traders, Clients, Observers, and Admins
- **Institutional IA** -- Configuration matrix, site map with role-based views, and document links

## Project Structure

```
index.html   # The entire knowledge base (single-page app)
CLAUDE.md    # Project context for Claude Code

```

## Making Changes

1. All content lives in `index.html`
2. Create a feature branch for significant changes
3. Merge to `main` -- GitHub Pages redeploys automatically
